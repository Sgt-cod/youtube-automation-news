Política de Privacidade

Última atualização: Janeiro de 2025

1. Informações que Coletamos

Este aplicativo é uma ferramenta de automação para publicação de conteúdo. Não coletamos dados pessoais de usuários finais.

2. Uso do TikTok

Utilizamos a API oficial do TikTok apenas para publicar vídeos em nossa própria conta. Não acessamos dados de terceiros.

3. Armazenamento de Dados

Os vídeos são gerados automaticamente e publicados diretamente nas plataformas. Não armazenamos dados de usuários.

4. Compartilhamento

Não compartilhamos dados com terceiros.

5. Contato

Para questões sobre privacidade, entre em contato através do nosso canal.
